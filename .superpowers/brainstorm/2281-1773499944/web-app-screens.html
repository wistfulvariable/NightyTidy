NightyTidy Web App — Screen Map

7 main screens. Click to select the ones you'd like to discuss further.

1

Landing / Login

nightytidy.com — marketing hero + "Sign in with GitHub" button. After auth, redirects to Dashboard. First-time users see agent install instructions.

2

Dashboard (Home)

Agent status (connected/disconnected) at top. Project cards showing last run, next scheduled run, quick-launch button. Active runs panel with live progress. Run queue with drag-to-reorder. Webhook feed showing recent events from all projects.

3

Project Detail

Click a project card to see: Run history table (date, steps, pass/fail, cost, duration). Step presets (save/load step combos). Schedule config (cron picker). Webhook config (add Slack/Discord/custom URLs). Analytics charts (cost trend, improvement rate, most valuable steps). Start New Run button.

4

Run Setup

Step selector — checkboxes for all 33 steps, grouped by category (Testing, Code Quality, Ops, Frontend, Infrastructure, Strategic). Step descriptions on hover. Load preset dropdown. Save as preset option. Timeout setting. Estimated cost range based on history. Smart recommendations badge on suggested steps. "Start Run" button.

5

Live Run View

Progress bar with step count. Current step with summarized activity feed ("Editing README.md... Adding tests to utils.test.js..."). Toggle to raw output panel (full Claude Code stream). Step list with checkmarks/X/spinner. Live cost counter. Pause / Skip Step / Stop Run buttons. Rate-limit overlay with countdown + "Resume Now".

6

Run Results

Summary stats (steps passed/failed, total cost, duration). Diff viewer — syntax-highlighted file-by-file diff of all changes. Step detail cards (expand to see what each step did). Action buttons: "Merge to main", "Create GitHub PR", "Rollback Everything", "Download Report". Full report (rendered NIGHTYTIDY-REPORT markdown).

7

Settings

Account — GitHub profile, sign out. Agent — connection info, reconnect instructions. Defaults — default timeout, default steps, log level. Notifications — global webhook endpoints, notification preferences. Appearance — dark/light theme toggle.
