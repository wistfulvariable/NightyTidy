Welcome to NightyTidy

AI-powered overnight codebase improvement. Pick what to improve, let Claude Code do the work, wake up to cleaner code.

⚙

How It Works

Choose from 43 improvement steps — documentation, testing, security, performance, and more. NightyTidy runs each one through Claude Code and generates a detailed report of everything that changed.

🔒

Built-In Safety Net

Every run creates a git branch and a safety tag before touching your code. If you don't like the changes, one command undoes everything. Your main branch stays untouched until you're ready to merge.

🛠

One Project at a Time

NightyTidy runs one session at a time — don't open multiple windows for different repos. Finish one run before starting the next. This prevents conflicts and keeps everything clean.

⚡

Token Usage

A full 43-step run is very token-heavy. You'll almost certainly need the Max plan with plenty of usage left. If you're not on Max, run 8–10 steps at a time and wait for your limits to reset between batches.

🌙

Ready to Go

Select a project folder, pick your steps, and let NightyTidy work overnight. If you hit a rate limit mid-run, it pauses automatically — you can resume later right where you left off.
